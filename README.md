## 1. 개요

- **핵심결론 요약**
  - 고객여정지도를 통해 주요 지점에서의 주요 이탈 지점을 명확하게 식별하고, 각 지점에 맞는 전략적인 액션을 사용하여 Google Merchandise Store의 고객의 접근을 늘리고, 구매 전환율을 유의미하게 개선할 수 있게 만든다
    
    **Step 1. 방문하기 → 상품보기**
    
    - **핵심 인사이트: 상품보기 전환율이 높은 [유입경로:referral] 사용자의 트래픽 비중을 늘린다**
    
    **Step 2. 상품보기 → 상품선택**
    
    - **핵심 인사이트: 핵심 인사이트: 사용자의 체류시간을 늘려 상품선택 전환율을 높인다**
    
    **Step 3. 상품선택 → 장바구니 담기**
    
    - **핵심 인사이트: 재구매 사용자는 다양한 카테고리를 탐색하므로, 다양한 카테고리 상품을 보여주는 추천시스템을 만든다**
    
    **Step 4. 장바구니 담기 → 결제**
    
    - **핵심 인사이트: 장바구니 담기 후 24시간 내 구매 전환을 유도한다**
    
    
- **분석 목적과 기대효과**
    1. Google Merchandise Store의 전반적 전환율 및 매출 KPI 모니터링
    2. 신규 고객과 기존 고객 간 구매 행동 비교 및 차이 도출
    3. 구글 브랜드 상품 재구매 고객 및 충성 고객 파악
    4. 상품 탐색 → 장바구니 담기 → 구매로 이어지는 단계별 이탈 및 전환율 분석

## 4. 분석 방법론

- **가설 설정**
    - **1순위 : 방문하기 -> 상품보기**
        
        **가설1 : 동일한 세션 내에 유입경로가 referral인 경우가 다른 유입경로보다 상품보기로의 전환율이 높아질 것이다.**
        
        **가설2 : 체류시간이 길어질수록 상품보기의 전환율이 높아질 것이다**
        
    - **2순위 : 장바구니 담기 -> 결제**
        
        **가설3 : 장바구니에 상품이 담긴 시간을 기준으로, 24시간 이내 구매 전환율이 50%보다 높다.**
        
        **가설4 : 서로 다른 날짜에 반복 구매한 재구매자는 하루 동안만 구매한 단일구매자보다 더 다양한 카테고리의 상품을 구매할 것이다.**
        
    - **3순위: 상품보기 -> 상품탐색**
        
        **가설5 : 신규 1회 방문자가 다른 사용자 분류보다 탐색 전환율이 낮을 것이다**
        
    - **4순위 : 상품탐색 -> 장바구니담기**
        
        **가설6: 세션당 상품 탐색 수가 많을수록 장바구니에 담길 확률이 높다**
        
        **가설7 : 유저타입에 따라 장바구니 전환율이 다를 것이다**
        
- **ML 모델링 / 통계 분석**: **로지스틱 회귀, XGBoost,LightGBM. RandomForest, CatBoost**
- **검증 절차**
  - 구매 데이터가 비구매에 비해 상대적으로 적은 불균형 데이터
  - KMeans 기반의 언더샘플링을 활용해 구매와 비구매 데이터를 1:1 비율로 재구성한 뒤 모델링을 진행
    
    
    | 모델 | F1-score | Accuracy |

    | Logistic Regression | 0.5349 | 0.6699 |
    | XGBoost | **0.7713** | 0.8593 |
    | LightGBM | 0.7605 | 0.8568 |
    | Random Forest | 0.7658 | 0.8603 |
    | **CatBoost** | **0.7713** | **0.8629** |
